# Hotel Reservation System

A comprehensive hotel reservation system built with JavaFX and Scene Builder. This application provides a user-friendly interface for managing hotel room bookings, customer information, and reservation details.

## 📋 Project Overview
This Hotel Reservation System is a desktop application developed using JavaFX with FXML for the user interface design. The system implements CRUD (Create, Read, Update, Delete) operations for efficient hotel room and reservation management.

## ✨ Features

### Room Management

- Add new rooms with details (room number, type, price, capacity)

- View all available rooms

- Update room information

- Delete rooms from the system

- Filter rooms by type and availability status

### Reservation Management

- View all reservations with customer details

- Cancel/delete reservations

### Customer Management

- Store customer information

- Manage customer details

## 🛠️ Technologies Used

- Java - Core programming language

- JavaFX - GUI framework

- Scene Builder - FXML UI design tool

- FXML - XML-based UI markup language

- CSS - Styling the application interface

- MySQL/SQLite - Database for data persistence 

## 🎨 UI Design
- Built with Scene Builder for visual design

- FXML files for clean separation of UI and logic

- CSS styling for consistent look and feel

- Responsive layout design

- User-friendly forms and tables
